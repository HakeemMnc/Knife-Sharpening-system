  3. InitiateCheckout - Tracked when payment form is displayed
  4. AddPaymentInfo - Tracked when user submits payment form
  5. Purchase - Tracked when payment is completed successfully

Testing Instructions:

  1. Visit http://localhost:3005
  2. Open browser Developer Tools (F12) and go to Console tab
  3. Look for "Meta Pixel tracked:" messages in console
  4. Start typing in the first name field - should see "Lead" event
  5. Complete the booking form and proceed to payment - should see "InitiateCheckout" event
  6. Submit payment form - should see "AddPaymentInfo" event
  7. Complete payment - should see "Purchase" event
